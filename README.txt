                          能　量　帳
                  Money  ·  Energy  ·  Ledger
═══════════════════════════════════════════════════════════════

                記錄金錢，也記錄你的生命流向。


  錢不是冷冰冰的數字，
  它是你時間、心力、選擇與願望的流動。
  透過每一筆收支，看見自己真正把能量交給了什麼。

  記帳不是檢討自己，而是溫柔地看見自己。



───────────────────────────────────────────────────────────────
  一　·　這個工具是什麼
───────────────────────────────────────────────────────────────

  這不是一般的記帳 App。

  一般工具問：「你花了多少錢？花在哪裡？」

  能量帳問的是：

      · 這筆錢讓你的能量增加，還是減少？
      · 這筆支出是滋養你，還是在消耗你？
      · 這筆收入是穩定的，還是壓力換來的？

  它是一面鏡子。



───────────────────────────────────────────────────────────────
  二　·　核心概念　｜　五行 + 霧
───────────────────────────────────────────────────────────────

  每一筆支出，都歸入一個「能量元素」：

    ┌─────────┬──────────────┬──────────────────────────┐
    │  元素   │  能量屬性    │  典型支出                │
    ├─────────┼──────────────┼──────────────────────────┤
    │  水     │  滋養、修復  │  好好吃飯、休息、療癒    │
    │  木     │  成長        │  學習、書籍、課程        │
    │  火     │  創造、連結  │  工具、作品、人際聚會    │
    │  土     │  必要        │  房租、保險、水電日用品  │
    │  金     │  布施        │  捐款、供養、助人        │
    │  霧     │  消耗、補償  │  衝動購物、壓力下犒賞    │
    └─────────┴──────────────┴──────────────────────────┘

  收入則歸於「金」（流通）。

  你給每筆錢一個能量屬性，久了就會看見——
  你的人生重心，到底落在哪些方向。



───────────────────────────────────────────────────────────────
  三　·　四個分頁　｜　從日常記錄到月度反思
───────────────────────────────────────────────────────────────


  【記　錄】　日常的入口

      · 快速記一筆
        金額　·　類別　·　能量屬性　·　當下狀態
        備註（事實）　·　一句覺察（內心的聲音）

      · 今日能量小日記
        標題下方會自動顯示今天的日期
        三個提示句可一鍵起頭：
          ✎ 今天最滋養我的一筆支出是……
          ✎ 今天我花錢時的主要狀態是……
          ✎ 今天我想感謝自己的是……
        一句話也好，不寫也沒關係。

      · 日記紀錄
        過往的小日記，可編輯、可刪除。
        預設顯示最近 5 則，更多用「顯示全部」展開。

      · 最近的流動
        最近 12 筆交易。
        如果該筆有「一句覺察」，會用金色斜體
        在那筆下方顯示。


  【流　向】　本月的全貌

      · 本月財氣        收入　·　支出　·　結餘
      · 金錢能量分數    滋養加分、消耗扣分（× 1.2 倍權重）
      · 支出的能量光譜  五行 + 霧的比例橫條圖
      · 一句洞察        根據資料模式自動生成的觀察


  【圖　鑑】　六種能量的本月分布

      水　·　木　·　火　·　土　·　金　·　霧
      每一張卡片顯示：金額、占比、典型用途。


  【回　顧】　月度反思的儀式

      · 翻月導覽      可前後翻看任何有資料的月份
      · 詩意總結卡    根據主導元素自動命名這個月
      · 本月數字
      · 金錢能量分數
      · 真正滋養你的支出　Top 3
      · 可能來自壓力的支出
          - 總額與占比
          - 最常在哪種「狀態」下發生
          - 集中在週幾、那天合計多少
          - 列出 Top 5 筆
      · 本月能量覺察
          - 寫日記天數
          - 帶覺察的支出筆數
          - 最常的能量／狀態
          - 最近 3 則小日記
      · 下個月的提示
          根據 pattern 自動生成 2~4 個反思問句
      · 完成這月回顧
          點下後本月光點熄滅。

      自動觸發　·　每月最後一天到隔月前 7 天，
      「回顧」分頁上會浮現一個微微脈動的金色光點。



───────────────────────────────────────────────────────────────
  四　·　怎麼開始用
───────────────────────────────────────────────────────────────

  1. 把 energy-ledger.html 用瀏覽器打開即可，無需安裝。

  2. 第一次打開：直接從「快速記一筆」開始。
     先不用追求完美，記下三筆就會看到雛形。

  3. 每筆建議至少填：
        - 金額
        - 類別
        - 能量屬性    （這是靈魂，請選一個）
     其餘（當下狀態、備註、一句覺察）皆選填。

  4. 一天結束時，可以在「今日能量小日記」留一句話。
     提示句點一下就會自動填入起頭。

  5. 每月最後一天，看看「回顧」分頁。
     等光點亮起的那天，就是這個月的能量結帳日。



───────────────────────────────────────────────────────────────
  五　·　資料儲存與隱私
───────────────────────────────────────────────────────────────

  · 所有資料只儲存在你開啟這個工具的瀏覽器／環境中。

  · 不會上傳、不會送到任何伺服器，也沒有帳號系統。

  · 三個獨立的儲存 key：
        energy_ledger_txns          交易紀錄
        energy_ledger_journals      每日小日記
        energy_ledger_reviewed      已完成回顧的月份

  · 「圖鑑」分頁底下的「清空所有紀錄」會一次清掉以上全部。
     此動作無法復原，會跳出確認對話框。

  · 因為是純前端 + 在地儲存，請避免：
        - 清掉這個瀏覽器的網站資料
        - 換瀏覽器或裝置時，資料不會自動跟著走

  · 想要備份？目前需手動。未來可加匯出 JSON / CSV。



───────────────────────────────────────────────────────────────
  六　·　設計上刻意的選擇
───────────────────────────────────────────────────────────────

  這個工具有意識地「不做」很多事，因為簡單比豐富珍貴：

  ✗ 不做連續打卡天數
        日記不是任務，沒有「中斷」這回事。

  ✗ 不做未寫日記提醒
        沒寫就是沒寫，那也是一種狀態。

  ✗ 不做嚴格的預算或目標
        記帳是觀察，不是審判。

  ✗ 不做社群分享
        錢與內心是私事。

  ✗ 不在資料不足時硬產報告
        沒紀錄的月份就溫柔地說「先去記錄」。

  ✗ 不用責備語氣
        看到「衝動消費」也不會跳警告，
        只是放在「霧」裡，讓你自己看見。

  做的選擇：

  ✓ 一句覺察 vs 備註　分開兩個欄位
        備註寫事實（買甜點），覺察寫感受（今天有點累）。
        不混為一談。

  ✓ 提示句而非空白
        空白頁讓人卻步，提示句讓人有起點。

  ✓ 資料驅動的詩意
        「霧氣稍重的一個月」這種標題，
        是根據實際比例自動選的，不是隨機文案。

  ✓ 每月最後一天才提醒
        平常不打擾，月底才出現光點，
        像月相一樣。



───────────────────────────────────────────────────────────────
  七　·　關於「能量分數」是怎麼算的
───────────────────────────────────────────────────────────────

  每一筆支出，按元素加減分（每 NT$1,000 = 1 分）：

        水（滋養／修復）   + 1.0×
        木（成長）         + 1.0×
        火（創造／連結）   + 1.0×
        金（布施）         + 1.0×
        土（必要）           0   （中性，不加不減）
        霧（消耗／補償）   - 1.2×（權重稍重，因為這類消費
                                  通常表示其他層面在透支）

  分數區間對應的描述：

        ≥ +30        能量飽滿，方向清明
        +10 ~ +29    穩定累積中
          0 ~ +9     平衡之中
        -1 ~ -15     需要留意流失方向
          ≤ -16      能量正在外漏，停下來看看

  這個分數不是「你過得好不好」的評分，
  是「你的錢的去向，跟你想要的方向，是否一致」的提示。

  八　·　未來可能會加的
───────────────────────────────────────────────────────────────

  · 事後感受評分（+2 / +1 / 0 / -1 / -2）

  · 夢想基金、修行布施紀錄

───────────────────────────────────────────────────────────────
  九　·　技術備註
───────────────────────────────────────────────────────────────

  · 純前端　·　單一 HTML 檔案
        HTML + CSS + 原生 JavaScript，無框架。

  · 字型
        Noto Serif TC          中文襯線
        Cormorant Garamond     拉丁文編輯襯線
        透過 Google Fonts 載入。

  · 持久化
        使用 window.storage（artifact 持久化儲存 API）
        非 localStorage，跨 session 自動保存。

  · 瀏覽器相容
        現代瀏覽器（Chrome / Safari / Firefox / Edge）皆可。
        不支援 IE。

  · 行動裝置
        已針對 460px 以下窄螢幕做斷點優化，
        圖鑑會從兩欄切換為單欄。

  · 沒有外部依賴（除字型），離線狀態下也能運作
        （字型可能會 fallback 到系統字）。
